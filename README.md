# Development
## Getting Started

```bash
npm run dev
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Anki/AnkiConnect

AnkiConnect settings -> add `"http://localhost:3000"` to `"webCorsOriginList"`

example:
```json
    "webCorsOriginList": [
        "http://localhost",
        "http://localhost:3000"
    ]
```
